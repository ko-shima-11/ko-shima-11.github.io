シフト自動生成システム（バックエンドのみ）

・特定の期間の間、登録されたシフト形態に合わせてシフトを自動生成するシステムのバックエンドを開発しました。
・自動生成の部分では充足可能性判定問題を解くツールであるSMTソルバー(Z3)を使用し、効率的なシフト生成を実現しています。
・必ず守るべき制約条件と、可能な限り守りたい制約条件を分けて設定できるようにしており、柔軟なシフト生成が可能です。

使用技術：Python, Z3 SMT Solver

ポモドーロ特化タイマー

・作業時間と休憩時間を設定し、タイマーが終了すると通知を送る機能を備えています。
・シンプルで使いやすいインターフェースを提供し、集中力を高めることを目的としています。
・時計のデザインとして、デジタル、アナログ、そして砂時計の3種類を選択可能です。
・またカラーも２種類から選択可能です。

使用技術：Next.js 14 / React 18 / TypeScript

倉庫内管理システム

・倉庫内のユニットに対して、在庫管理や入出庫管理を行うシステムを開発しました。
・ユーザーはウェブインターフェースを通じて、在庫の追加、削除、更新を行うことができます。
・追加機能では、QRコードからの読み込みに対応しており、迅速な入出庫処理が可能です。
・また、入出庫の履歴を記録し、レポートを生成する機能も備えています。
・このシステムは実際に企業と協力しながら、チームで開発を進めました。
・詳しくは下記のページをご覧ください。

年賀状補助システム

・年賀状の宛名印刷を補助するシステムを開発しました。

3Dリバーシ

・6×6×3の多層盤面で遊ぶ、Web版の3Dリバーシを開発しています。中央レイヤーは6×6、外側レイヤーは中央寄り4×4を有効マスとする独自ルールです。
・通常の縦横斜めに加えて、レイヤー方向も含めた3D方向の置ける場所判定、石反転、パス判定、終局判定をTypeScriptのルールエンジンとして分離しました。
・4手ごとに外側レイヤーへ石を自動配置するイベントや、Easy / Normal / 最強級チャレンジのCPU対戦を実装しています。
・盤面はレイヤー表示と3D全体表示を切り替えられ、Three.jsで回転・ズームしながら立体盤面を確認できます。
・公開URLは下記から確認できます。
